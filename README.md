# web-game
A web-based game that also saves &amp; tracks user scores!
Uses JavaScript for gameplay mechanics.
Uses HTML for webpages displaying.
Uses an SQLite to securely store usernames, hashed passwords & high scores.
Relies on Flask framework from Python to hold it all together, including management of logins and logouts.
